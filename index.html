BIOS

basic input output System

是一组固化到计算机主板上一个ROM芯片上的程序，保存着计算机最重要的基本输入输出的程序、开机后自检和系统自启动。
可以从CMOS中读写系统设置的基本信息。
主要功能是为系统提供最底层、最直接的硬件设置和控制。
一块主板性能优越与否，很大程度上取决于主板上的BIOS管理功能是否先进。
